反转链表

首先介绍最简单的反转链表

题目描述:

Reverse a singly linked list.

分析：

题目比较简单，可以使用迭代或者递归分别实现。

  • 迭代实现 原链表中的元素依次插入到新的链表中，作为新链表的第一个元素，这里需要注意的是，防止断链。
def reverseList(self, head):
    dummy = ListNode(-1)
    cur = None
    dummy.next = cur
    while head :
        q = head.next # 防止断链
        dummy.next = head
        head.next = cur
        cur = head
        head = q
    return dummy.next
