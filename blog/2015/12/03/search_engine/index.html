在给定测试集中，信息检索中通常采用正确率和召回率来度量系统的效果。 正确率是返回的结果中相关文档所占的比例；而召回率是返回的相关文档占所有相关文档的比例。

文本分类： 多项式模型 P(d|c) = P(|C), 其中是在d中出现的词项序列。 贝努利模型 p(d|c) = P(|C), 其中是一个M纬的布尔向量，表示每个词在文档d中存在与否。

在朴素贝叶斯模型中，有两个独立性假设，分别是条件独立性假设，即给定类别时，假设属性值之间是相互独立的。另一个是位置独立性假设，即词在文档中每个位置的出现概率是一样的。

web crawler

任何超文本采集器的基本处理如下:首先，设定一个或者多个URL为采集的种子集合，接着从种子集合中选择一个URL进行采集，然后对采集到的页面进行分析，并抽取出页面中的文本和连接，抽取出的文本输给文本索引器而抽取出的URL则加入到待采集URL池，任何时候URL池中放的都是所有代采集网页的URL。一开始，种子集合会放入到URL中，一旦某个URL被采集，就从URL池中删除这个URL。整个采集过程可以看成是Web图的遍历过程。

web crawler 架构

一个简单的采集器有多个模块构成，其中主要包括5个模块： - 待采集URL池，它包含了当前待采集的URL - DNS解析模块，它在URL抓取网页时用于确定其对应的Web服务器的IP地址 - 抓取模块，利用http协议返回某个URL对应的网页 - 分析模块，从采集到的网页中抽取文本及链接 - URL去重模块，确定某个抽取出的链接是否已在URL池中， 或者最近是否被抓取

倒排索引

lucence使用倒排数据结构的原因是：把文档中提取的语汇单元作为查询关键字，而不是将文档作为中心实体。倒排索引不是回答一个文档中包含了哪些单词，而是这个单词在哪些个文档中出现。

拼写校正

拼写校正的原则： 1. 对于一个拼写错误的查询，在其可能的正确拼写中，选择距离最近的一个。 2. 当两个拼写查询邻近度相等（或相近）是，选择更常见的那个。
