优先队列中的排序

优先队列中默认的是从大到小的序列，如果要实现从小到大的序列或者是自己定义的数据节点就需要自己实现一个比较函数。下面以对pair元素为例，具体说下优先队列中的比较函数的实现。

自定义节点

struct Pair
{
    pair<string, int> node1;
    bool operator<(const pair<char, int>& node2)
    {
        return node1.second < node2.second;
    }
    bool operator>(const pair<char, int>& node2)
    {
        return node1.second > node2.second;
    }
};
priority_queue<Pair, vector<Pair>, greater<Pair> > heap;

结构体运算符重载

struct MyStruct
{
    bool operator()(pair<string, int> n1, pair<char, int> n2)
    {
        return n1.second < n2.second;
    }
};
priority_queue<pair<string,int>, vector<pair<string, int>>, MyStruct> heap;
